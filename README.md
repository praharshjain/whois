# whois
Simple shell script to fetch whois information for a domain using zenity for UI.  
Usage
----------------
If zenity is not installed - 
```
$ sudo apt-get update
$ sudo apt-get install zenity
```
Make the file executable - 
```
$ chmod +x whois.sh
```  
Run - 
```
$ ./whois.sh
```
